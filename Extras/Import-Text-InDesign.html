InDesign Text Import Workflow

Importing Text While Keeping Formatting

To bring text into Adobe InDesign and keep basic formatting such as bold, italics, and relative font sizes:

  • Save the source text as a .docx or .rtf file.
  • In InDesign, choose File > Place.
  • Enable Show Import Options in the Place dialog.
  • In the import options, choose Preserve Styles and Formatting from Text and Tables, then place the text.

Using Apple Pages as the Source

If you write in Apple Pages and want to use that text in InDesign while preserving styling:

  • In Pages, choose File > Export To > Word….
  • Export the document as a Word (.docx) file.
  • Open InDesign and use File > Place on the exported .docx file with Show Import Options enabled.
  • Again choose Preserve Styles and Formatting from Text and Tables before placing.

Optional Style Mapping

For more control, you can map incoming Word styles to your InDesign paragraph and character styles in the import options dialog. This lets you keep emphasis like italics while conforming to your newspaper’s style sheet.
