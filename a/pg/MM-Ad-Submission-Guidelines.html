Technical Requirements for Ad Submission

Please review and follow the guidelines below when sending files:

Color & Black Usage

  • All color artwork and photos must be set to CMYK (not RGB).
  • All text should be 100% Black (K only) — do not use rich black or CMYK black for text. This helps prevent readability and registration issues.
  • Logos or photos intended to appear grayscale should be set to Grayscale, not CMYK.

Basic Pre-Press Production Checks

1. Document Setup

  • Ensure the document size matches the final printed size.
  • Confirm the color mode is CMYK.
  • Check image and graphic resolution (typically 300 dpi for print).
  • Verify that black text and solid black elements are set to 100% K only.

2. Color Check

  • Review color consistency throughout the document.
  • Check for spot colors or special inks and convert them to CMYK or 100% Black, if applicable.

3. Font Check

  • Ensure all fonts are embedded or outlined.

4. Image Check

  • Confirm all linked images are properly placed and high resolution.
